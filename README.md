# 📄 Disk & Ulimit Management

## 🚀 Overview
This project explains basic disk management and ulimit configuration in Linux (Ubuntu).  
It helps in monitoring disk usage and controlling system resource limits to avoid system failures.

---

## 🎯 Purpose
- Monitor disk usage
- Prevent disk full issues
- Control system resources using ulimit
- Improve system stability

---

## 📌 Scope
- Ubuntu-based systems
- Dev / Test environments
- Beginner DevOps use cases

---

## ⚙️ Prerequisites
- Ubuntu system
- sudo/root access
- Basic Linux knowledge
